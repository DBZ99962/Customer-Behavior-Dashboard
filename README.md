# 🛒 Customer Shopping Behavior Analysis

## 📌 Project Overview
This project analyzes customer shopping behavior using a dataset of 3,900 transactions across various product categories. The primary objective is to uncover actionable insights into spending patterns, customer segmentation, product preferences, and subscription behaviors to drive strategic business decisions. 

## 🛠️ Tools & Technologies Used
* **Data Cleaning & Preparation:** Python (Pandas)
* **Relational Database / Querying:** PostgreSQL
* **Data Visualization & Dashboarding:** Power BI
* **Reporting:** PowerPoint, PDF Documentation
